## 🔧 模型参数调优

### Temperature（温度）

**作用**：控制输出的随机性

**取值范围**：0.0 - 2.0

**推荐设置**：
- **数据分析**：0.2 - 0.3（更确定性）
- **投资建议**：0.3 - 0.5（平衡）
- **创意分析**：0.6 - 0.8（更多样性）

### Max Tokens（最大令牌数）

**作用**：控制输出长度

**推荐设置**：
- **简要分析**：500 - 1000
- **标准分析**：1000 - 2000
- **深度报告**：2000 - 4000

### Top P

**作用**：控制输出的多样性

**推荐设置**：
- **精确分析**：0.8 - 0.9
- **一般分析**：0.9 - 0.95
- **创意分析**：0.95 - 1.0

## 🚀 第一步：单股分析

### 1.1 进入分析页面

1. 登录系统
2. 点击左侧菜单"股票分析"
3. 进入分析页面

### 1.2 输入股票代码

**支持的格式**：

```
✅ 000001        # 6位代码
✅ 平安银行       # 股票名称
✅ 000001.SZ     # 带市场后缀
```

**示例**：
```
输入：000001
或者：平安银行
```

### 1.3 选择分析类型

**基础分析**：
- 适合：快速了解股票基本情况
- 时间：约30秒
- 内容：基本面 + 技术面概览

**深度分析**：
- 适合：重要投资决策
- 时间：约2-3分钟
- 内容：多维度深度分析 + 风险评估

**多智能体辩论**：
- 适合：重大投资决策
- 时间：约5-8分钟
- 内容：多个AI从不同角度辩论分析

### 1.4 点击"开始分析"

系统将自动：
1. 获取最新数据
2. 调用AI模型分析
3. 生成分析报告

## 📊 第二步：解读分析报告

### 2.1 报告结构

一份完整的分析报告包含：

```
📄 分析报告
├── 📌 执行摘要
│   ├── 综合评分（1-5星）
│   ├── 投资建议（买入/持有/卖出）
│   └── 核心观点
│
├── 📈 基本面分析
│   ├── 估值水平
│   ├── 盈利能力
│   ├── 成长性
│   └── 财务健康度
│
├── 📉 技术面分析
│   ├── 趋势判断
│   ├── 技术指标
│   └── 关键价位
│
├── ⚠️ 风险评估
│   ├── 主要风险
│   ├── 风险等级
│   └── 应对建议
│
└── 💡 投资建议
    ├── 操作建议
    ├── 目标价位
    └── 持有期限
```

### 2.2 关键指标解读

#### 综合评分

```
⭐⭐⭐⭐⭐ (5星) - 强烈推荐
⭐⭐⭐⭐   (4星) - 推荐
⭐⭐⭐     (3星) - 中性
⭐⭐       (2星) - 不推荐
⭐         (1星) - 强烈不推荐
```

#### 投资建议

```
🟢 买入    - 当前价位具有投资价值
🟡 持有    - 已持有可继续持有，未持有观望
🔴 卖出    - 建议减仓或清仓
```

#### 风险等级

```
🟢 低风险   - 风险可控，适合稳健投资者
🟡 中等风险 - 需要一定风险承受能力
🔴 高风险   - 仅适合激进投资者
```

### 2.3 示例报告解读

**示例：平安银行（000001）分析报告**

```markdown
## 执行摘要

**综合评分**：⭐⭐⭐⭐ (4星)
**投资建议**：🟢 买入
**目标价位**：14.50元
**风险等级**：🟡 中等风险

**核心观点**：
平安银行当前估值处于历史低位，PE 5.2远低于行业平均6.5，
具有明显的估值修复空间。公司零售银行业务转型成效显著，
ROE保持在10%以上。主要风险在于不良贷款率略高于同业，
需关注资产质量变化。

## 基本面分析

### 估值水平 ⭐⭐⭐⭐⭐
- PE 5.2 vs 行业平均 6.5：**低估约20%**
- PB 0.6：**破净，估值吸引力强**
- 历史PE区间：4.5-8.5，当前处于**低位**

### 盈利能力 ⭐⭐⭐⭐
- ROE 10.5%：**行业中等水平**
- 净利润增长率 8.5%：**稳健增长**
- 净息差 2.3%：**略有收窄压力**

### 成长性 ⭐⭐⭐
- 营收增长率 8.5%：**稳健**
- 零售业务占比提升：**转型成效显现**
- 数字化转型：**长期增长动力**

### 财务健康度 ⭐⭐⭐
- 资本充足率 12.5%：**符合监管要求**
- 不良贷款率 1.2%：**略高于同业**
- 拨备覆盖率 200%：**风险缓冲充足**

## 技术面分析

### 趋势判断
- 短期：震荡
- 中期：上升通道
- 长期：底部区域

### 技术指标
- MACD：金叉信号 ✅
- RSI：55（中性）
- 均线：站上60日均线 ✅

### 关键价位
- 支撑位：11.80元
- 当前价：12.50元
- 压力位：13.20元

## 风险评估

### 主要风险
1. **信用风险**：不良贷款率1.2%，高于行业平均1.0%
2. **政策风险**：金融监管趋严，影响业务扩张
3. **市场风险**：利率波动影响净息差

### 风险等级：🟡 中等

### 应对建议
- 控制仓位在10-15%
- 设置止损位10.50元
- 关注季度财报，动态调整

## 投资建议

### 操作建议
**🟢 买入**

### 理由
1. 估值处于历史低位，安全边际高
2. 基本面稳健，盈利能力良好
3. 技术面出现积极信号
4. 风险可控，风险收益比合理

### 目标价位
- 短期目标：13.50元（+8%）
- 中期目标：14.50元（+16%）

### 持有期限
6-12个月

### 建议仓位
10-15%（根据个人风险偏好调整）

### 止损位
10.50元（-16%）
```

**如何使用这份报告**：

1. **先看执行摘要**：快速了解核心观点
2. **重点看基本面**：判断长期投资价值
3. **参考技术面**：把握买入时机
4. **必看风险评估**：了解潜在风险
5. **结合自身情况**：做出独立判断

## 🗣️ 第三步：使用多智能体辩论

### 3.1 什么时候使用辩论？

**适合场景**：
- 💰 大额投资决策
- 🤔 对股票有疑虑
- 📊 需要多角度分析
- ⚖️ 想听不同观点

### 3.2 如何使用辩论功能？

1. 在分析页面选择"多智能体辩论"
2. 输入股票代码
3. 点击"开始辩论"
4. 等待3-5分钟

### 3.3 辩论报告示例

```markdown
## 辩论主题：平安银行（000001）是否值得买入？

### 第1轮：初始观点

**乐观派（基本面分析师）**：
"PE 5.2，严重低估，建议买入。
银行股历史上PE低于5.5时，未来一年平均涨幅达25%。"

**谨慎派（风险管理员）**：
"不良贷款率1.2%偏高，且净息差持续收窄，
盈利增长面临压力，建议观望。"

### 第2轮：深入辩论

**乐观派反驳**：
"不良贷款率1.2%虽略高，但：
1. 拨备覆盖率200%，风险可控
2. 公司加大不良处置力度，趋势向好
3. 零售转型降低对公业务风险
估值低估的幅度远大于风险溢价。"

**谨慎派反驳**：
"估值低有其原因：
1. 银行业整体面临净息差收窄
2. 经济下行压力影响资产质量
3. 监管趋严限制业务扩张
低估不等于会涨，可能继续低估。"

### 第3轮：达成共识

**投资经理综合**：
"双方观点都有道理。综合判断：

**优势**：
- 估值确实有吸引力
- 基本面相对稳健
- 技术面出现积极信号

**风险**：
- 行业面临挑战
- 资产质量需关注
- 短期催化剂不明确

**最终建议**：
- 评级：🟢 谨慎买入
- 仓位：10-15%（不要重仓）
- 止损：10.50元
- 策略：分批建仓，动态调整"
```

**辩论的价值**：
- 看到不同角度的观点
- 理解风险和机会
- 做出更理性的决策

## ⚙️ 第四步：配置LLM模型

### 4.1 进入设置页面

1. 点击左侧菜单"设置"
2. 选择"LLM配置"

### 4.2 选择模型

**推荐配置**：

**预算有限**：
```
主力模型：GPT-3.5-Turbo
备用模型：通义千问
```

**预算充足**：
```
主力模型：GPT-4-Turbo
深度分析：GPT-4
```

**中文优化**：
```
主力模型：文心一言 4.0
备用模型：GPT-4-Turbo
```

### 4.3 配置API密钥

1. 获取API密钥（从模型提供商官网）
2. 在设置页面输入API密钥
3. 点击"测试连接"
4. 保存配置

### 4.4 调整参数

**Temperature（温度）**：
```
数据分析：0.2-0.3（更确定）
投资建议：0.3-0.5（平衡）
```

**Max Tokens（最大长度）**：
```
快速分析：1000
标准分析：2000
深度分析：4000
```

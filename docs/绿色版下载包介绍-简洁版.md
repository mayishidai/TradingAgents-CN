# TradingAgents-CN v1.0.0-preview 绿色版

## 软件简介

TradingAgents-CN 是一款基于多智能体大语言模型的中文金融交易决策系统，专为中文用户优化，提供完整的 A股/港股/美股分析能力。

绿色版特点: 免安装 | 开箱即用 | 一键启动 | 数据本地化

---

## 核心功能

**多智能体协作分析**
- 5 大专业智能体: 基本面分析师、技术分析师、新闻分析师、风险管理师、投资组合经理
- 智能协作决策: 多角度分析，综合生成投资建议
- 专业报告导出: 支持 Markdown/Word/PDF 格式

**全面市场覆盖**
- A股: 沪深主板、创业板、科创板、北交所
- 港股: 香港联交所上市股票
- 美股: 纳斯达克、纽交所上市股票

**多大模型支持**
- 国际模型: OpenAI GPT-4、Claude、Gemini
- 国产模型: 智谱 GLM-4、通义千问、文心一言、Kimi、DeepSeek
- 本地模型: Ollama 本地部署

**实用功能**
- 单股深度分析
- 批量股票分析
- 智能股票筛选
- 自选股管理
- 实时行情监控
- 历史分析记录

## 系统要求

**最低配置**
- 操作系统: Windows 10 64位
- 处理器: Intel Core i3
- 内存: 4GB RAM
- 硬盘: 2GB 可用空间
- 网络: 需要互联网连接

**推荐配置**
- 操作系统: Windows 11 64位
- 处理器: Intel Core i5 或更高
- 内存: 8GB RAM 或更多
- 硬盘: 5GB 可用空间

## 快速开始

**1. 下载解压**
- 下载绿色版压缩包
- 解压到任意目录（建议路径不含中文）

**2. 启动系统**
- 双击 `启动.bat`
- 等待服务启动（约 30 秒）
- 浏览器自动打开系统界面

**3. 初始配置**
- 注册管理员账号
- 配置大模型 API 密钥（推荐：智谱 GLM-4）
- 配置数据源（可选：Tushare Token）

**4. 开始分析**
- 输入股票代码（如：000001）
- 选择分析类型
- 点击"开始分析"

## 常见问题

**Q: 启动失败，提示端口被占用？**

A: 双击 `配置.bat`，选择"修改端口配置"，修改为其他可用端口。

**Q: 分析失败，提示 API 调用失败？**

A: 检查大模型配置，确认 API 密钥正确且有足够额度。

**Q: 如何备份数据？**

A: 停止系统后，复制 `data` 目录到安全位置即可。

**Q: 如何升级新版本？**

A: 备份 `data` 和 `config` 目录，下载新版本后复制回来。

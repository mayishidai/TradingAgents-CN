## 软件简介

TradingAgents-CN 是一款基于多智能体大语言模型的中文金融交易决策系统，专为中文用户优化，提供完整的 A股/港股/美股分析能力。

### 核心特性

**多智能体协作分析**
- 基本面分析师: 深度解读财务报表、估值指标、行业地位
- 技术分析师: 精准计算技术指标、识别趋势形态
- 新闻分析师: 智能抓取和分析市场新闻、舆情监控
- 风险管理师: 全面评估投资风险、制定风控策略
- 投资组合经理: 综合决策、生成专业投资建议

**企业级功能**
- 智能股票筛选: 基于多维度指标的股票筛选和排序系统
- 自选股管理: 个人自选股收藏、分组管理和跟踪功能
- 个股详情页: 完整的个股信息展示和历史分析记录
- 批量分析功能: 支持多只股票同时分析，提升工作效率
- 报告导出功能: 支持 Markdown/Word/PDF 多格式专业报告导出
- 实时通知系统: 实时跟踪分析进度和系统状态

**多市场支持**
- A股市场: 完整支持沪深主板、创业板、科创板、北交所
- 港股市场: 支持香港联交所上市股票
- 美股市场: 支持纳斯达克、纽交所上市股票

**多大模型支持**
- 国际模型: OpenAI GPT-4、Claude、Gemini
- 国产模型: 智谱 GLM-4、通义千问、文心一言、Kimi、DeepSeek
- 本地模型: Ollama 本地部署支持

## 快速开始

**第一步：启动系统**

1. 双击 `启动.bat`
2. 等待命令行窗口显示 "系统启动成功"
3. 浏览器自动打开 `http://localhost:3000`

**第二步：初始化配置**

1. 注册账号
   - 首次使用需要注册管理员账号
   - 填写用户名、邮箱、密码

2. 配置大模型
   - 进入 "系统设置" → "大模型配置"
   - 添加至少一个 LLM 提供商的 API 密钥
   - 推荐：智谱 GLM-4（国产免费额度）或 OpenAI GPT-4

3. 配置数据源（可选）
   - 进入 "系统设置" → "数据源配置"
   - 配置 Tushare Token（推荐，需注册）
   - 或使用 AKShare（免费，无需注册）

**第三步：开始分析**

1. 单股分析
   - 点击 "单股分析"
   - 输入股票代码（如：000001）
   - 选择分析类型（快速/标准/深度）
   - 点击 "开始分析"

2. 批量分析
   - 点击 "批量分析"
   - 输入多个股票代码（每行一个）
   - 选择分析类型
   - 点击 "开始分析"

3. 股票筛选
   - 点击 "股票筛选"
   - 设置筛选条件（市值、PE、ROE 等）
   - 点击 "筛选"
   - 查看筛选结果

## 常见问题

**Q1: 启动失败，提示端口被占用？**

A: 可能是端口冲突，请按以下步骤解决：

1. 双击 `配置.bat` 运行配置向导
2. 选择 "修改端口配置"
3. 修改为其他可用端口（如 3001、8001）
4. 重新启动系统

详细说明：[端口配置说明](https://mp.weixin.qq.com/s/o5QdNuh2-iKkIHzJXCj7vQ)

**Q2: 分析失败，提示 "LLM API 调用失败"？**

A: 请检查 LLM 配置：

1. 进入 "系统设置" → "大模型配置"
2. 确认 API 密钥正确
3. 确认账户有足够的额度
4. 测试 API 连接

**Q3: 无法获取股票数据？**

A: 请检查数据源配置：

1. 如果使用 Tushare，确认 Token 正确且有权限
2. 如果使用 AKShare，确认网络连接正常
3. 尝试切换到其他数据源

**Q4: 如何备份数据？**

A: 数据备份非常简单：

1. 停止系统（双击 `停止.bat`）
2. 复制整个 `data` 目录到安全位置
3. 需要恢复时，将备份的 `data` 目录覆盖回来

**Q5: 如何升级到新版本？**

A: 升级步骤：

1. 备份 `data` 和 `config` 目录
2. 下载新版本绿色包
3. 解压到新目录
4. 将备份的 `data` 和 `config` 目录复制到新版本
5. 启动新版本
